# Screenshot to Eagle - 使用指南

## 快速开始

### 1. 安装与配置

#### 第一步：确保 Eagle 正在运行
- 打开 Eagle 应用
- 确保 Eagle Web API 已启用（默认启用）

#### 第二步：配置扩展
1. 打开 Raycast
2. 搜索 "Screenshot to Eagle"
3. 按 `⌘ + ,` 打开扩展设置
4. 配置以下选项：

**基础配置**（可选）：
- Eagle API URL: `http://localhost:41595`（默认）
- Eagle API Token: 在 Eagle → 偏好设置 → 插件 → Web API 中查找

**可选配置**：
- 默认截图模式：选择你喜欢的截图方式
- 自动标签：输入逗号分隔的标签，如 `screenshot, raycast, design`
- 包含鼠标指针：是否在截图中显示鼠标
- 截图音效：是否播放快门声音
- 延迟时间：延迟截图模式的等待秒数

### 2. 选择保存文件夹（推荐）

1. 在 Raycast 中搜索 "Select Eagle Folder"
2. 浏览显示的文件夹列表：
   - **Current Selection**: 当前选中的文件夹
   - **Recent Folders**: 最近使用的文件夹
   - **All Folders**: 所有可用文件夹
3. 点击选择你想要保存截图的文件夹
4. 看到成功提示后，配置完成！

### 3. 开始截图

1. 在 Raycast 中搜索 "Shot to Eagle"
2. 或者设置快捷键（推荐：`⌘ + ⇧ + 4`）
3. 屏幕会进入截图模式
4. 根据你的配置模式进行截图：
   - **选择模式**：拖动选择区域
   - **窗口模式**：点击窗口
   - **全屏模式**：自动截取全屏
   - **延迟模式**：等待倒计时后选择区域
5. 截图完成后会自动上传到 Eagle
6. 看到 "✓ Saved to Eagle" 提示表示成功

## 高级技巧

### 技巧 1：使用快捷键
在 Raycast 设置中为 "Shot to Eagle" 命令设置快捷键：
- 推荐：`⌘ + ⇧ + 4`（替代系统截图）
- 或者：`⌘ + ⇧ + 5`

### 技巧 2：配置多个工作流
虽然当前版本只支持一个默认配置，但你可以：
1. 使用 "Select Eagle Folder" 快速切换目标文件夹
2. 在 Eagle 中使用智能文件夹自动分类
3. 在 Eagle 中批量修改标签

### 技巧 3：结合 Eagle 功能
- 在 Eagle 中为截图文件夹设置自动标签规则
- 使用 Eagle 的 OCR 功能提取截图中的文字
- 利用 Eagle 的相似图片搜索功能去重

### 技巧 4：延迟截图的妙用
使用延迟截图（Timed Mode）可以：
- 截取鼠标悬停效果
- 截取下拉菜单
- 截取弹出窗口
- 准备表情和姿势 😄

## 常见问题

### Q1: 截图后没有反应？
**A**: 检查以下几点：
- Eagle 是否正在运行？
- Eagle Web API 是否启用？
- 网络连接是否正常？
- 查看 Raycast 是否显示错误提示

### Q2: 如何找到 Eagle API Token？
**A**: 
1. 打开 Eagle
2. Eagle → 偏好设置（⌘ + ,）
3. 插件 → Web API
4. 复制 Token（格式如：`f4f16c3d-a969-4060-b6db-773918ef5cc7`）

### Q3: 可以更改截图文件名吗？
**A**: 当前版本使用固定格式：`Screenshot_YYYY-MM-DD_HH-mm-ss`
未来版本会支持自定义文件名模板。

### Q4: 截图会占用很多存储空间吗？
**A**: 
- PNG 格式，质量较高
- 建议定期清理不需要的截图
- Eagle 支持压缩功能

### Q5: 可以截取部分应用吗？
**A**: 
- 使用选择模式（Selection）可以截取任意区域
- 使用窗口模式（Window）可以截取整个窗口
- macOS 系统限制，某些安全应用可能无法截取

### Q6: 如何批量导入已有截图？
**A**: 
这个扩展专注于实时截图，批量导入建议：
1. 直接在 Eagle 中使用导入功能
2. 或使用 Eagle 的文件夹监控功能

## 快捷键参考

### macOS 系统截图快捷键
- `⌘ + ⇧ + 3`: 全屏截图
- `⌘ + ⇧ + 4`: 区域截图
- `⌘ + ⇧ + 4 + Space`: 窗口截图
- `⌘ + ⇧ + 5`: 截图工具

### 建议的 Raycast 快捷键
- Shot to Eagle: `⌘ + ⇧ + 4`（覆盖系统默认）
- Select Eagle Folder: `⌘ + ⇧ + E`

### 截图过程中
- `ESC`: 取消截图
- `Space`: 在区域选择和窗口选择之间切换
- 拖动时按住 `Space`: 移动选择区域
- 拖动时按住 `⇧`: 锁定宽高比
- 拖动时按住 `⌥`: 从中心扩展

## 性能优化

### 减少上传时间
- 使用有线网络而非 Wi-Fi
- 关闭不必要的后台应用
- 确保 Eagle 有足够的存储空间

### 减少存储占用
- 在 Eagle 中启用智能压缩
- 定期清理重复截图
- 使用 Eagle 的相似图片检测
